3 . Distribuciones de frecuencia

Introducción: Las distribuciones de frecuencia son herramientas fundamentales en estadística y análisis de datos para organizar y resumir información. Nos permiten visualizar cuántas veces aparece cada valor diferente dentro de un conjunto de datos.

Funcionamiento Las distribuciones de frecuencia son herramientas fundamentales en estadística y análisis de datos para organizar y resumir información. Nos permiten visualizar cuántas veces aparece cada valor diferente dentro de un conjunto de datos.

Funcionamiento Imagine un conjunto de datos con las edades de los estudiantes en una clase. Una distribución de frecuencia agruparía estas edades, contando cuántos estudiantes pertenecen a cada grupo de edad.

Existen diversas formas de presentar una distribución de frecuencia:

  • Tablas: Listan los valores únicos y sus frecuencias correspondientes.

  • Gráficos: Representan visualmente la información, como histogramas, polígonos de frecuencia o diagramas de sectores.

Utilidad

Las distribuciones de frecuencia son versátiles y ofrecen múltiples beneficios:

  • Identificar patrones: Revelan si los datos se agrupan alrededor de ciertos valores o si existen valores atípicos.
  • Comparar grupos: Permiten comparar las distribuciones de frecuencia de distintos grupos para identificar diferencias significativas.
  • Comprender la forma de los datos: Observar si los datos son simétricos, sesgados o presentan otra forma particular.
  • Base para análisis adicionales: Sirven como punto de partida para calcular medidas de tendencia central (media, mediana, moda) y de dispersión (varianza, desviación estándar).

3.1 Tipos de distribuciones de frecuencia

Frecuencia absoluta(ni):

¿Qué es? Simplemente cuenta cuántas veces aparece un valor específico dentro del conjunto de datos. - Ejemplo: Si en una clase hay 5 estudiantes con 20 años, la frecuencia absoluta para la edad de 20 años es 5. - Utilidad: Nos da una idea clara de la popularidad de cada valor.

Frecuencia relativa(hi):

  • ¿Qué es? Expresa la frecuencia absoluta como una proporción o porcentaje del total de datos. Se calcula dividiendo la frecuencia absoluta de un valor entre el número total de datos.

  • Ejemplo: Si hay 25 estudiantes en total, la frecuencia relativa para la edad de 20 años (con 5 estudiantes) sería 5/25 = 0.2 o 20%.

  • Utilidad: Permite comparar la frecuencia de valores entre diferentes conjuntos de datos, incluso si tienen tamaños distintos.

Frecuencia absoluta acumulada(Ni):

  • ¿Qué es? Suma las frecuencias absolutas hasta un cierto punto. Es decir, nos dice cuántos valores son menores o iguales a un valor específico.

  • Ejemplo: Si las edades en la clase son 20, 21, 22 y 23, la frecuencia acumulada para la edad de 21 años sería la suma de las frecuencias de 20 y 21 años.

  • Utilidad: Ayuda a entender la distribución de los datos y a determinar cuántos valores se encuentran por debajo de cierto límite.

Frecuencia relativa acumulada(Hi):

  • Qué es? Similar a la frecuencia absoluta acumulada, pero expresada como proporción o porcentaje. Se calcula dividiendo la frecuencia absoluta acumulada entre el número total de datos.

  • Ejemplo: Usando el ejemplo anterior, la frecuencia relativa acumulada para la edad de 21 años sería la proporción de estudiantes con 21 años o menos en relación al total de estudiantes.

  • Utilidad: Permite comparar la distribución de datos entre diferentes conjuntos y analizar la proporción de valores que se encuentran por debajo de cierto límite.

4.1 Asignar las categorías a las variables cualitativas y creacion de tablas de frecuencia.

Primero se creara una funcion de creacion de tablas con frecuencias de valor absoluto para variables de datos.

Luego se crearan las tablas en un objeto vinculado a las frecuencias calculadas, despues asigaremos un valor factor a las variables de la tabla y generaremos las tablas simplemente especificando la variable.

Por ultimo se da un valor invisible en las tablas generadas para ponerlas en las pestañas y volver a darle visibilidad, esto con la intension de un mejor orden y no tener la misma tabla 2 veces.

  • Edad
  • Sexo
  • Escolaridad
  • Nacimiento
  • Ingresos
  • Peso
  • Estatura
  • Partida
  • Motivo
  • Tiempo de visita
  • Visita anterior
  • Actividad
  • Gasto

# Necesario reescribir la base de la funcion pero ahora visible

crear_tabla_frecuencia <- function(variable, datos) {
  frec_abs <- table(datos[[variable]])
  frec_rel <- round(prop.table(frec_abs), 4)
  frec_rel_porcentaje <- frec_rel * 100
  
  tabla <- cbind(frec_abs, frec_rel, frec_rel_porcentaje)
  colnames(tabla) <- c("ni", "hi", "hi%")
  
  # Tabla ahora visible
  tabla %>% 
    kable(align = "c") %>%
    kable_styling(bootstrap_options = c("striped", "bordered", "hover", "condensed", "responsive"))
}
